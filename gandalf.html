Full-Stack Development and Commercialization of Gandalf KMS

Project information

  • Category: Full-Stack Web Development
  • Client: Reboot Conseil
  • Project date: July 2023 - January 2024
  • Technology: VueJS, NodeJS, C# .NET, EF Core, MSSQL, Azure DevOps, SCRUM

Project Overview

At Reboot Conseil, I had the distinct opportunity to be a pivotal contributor to the development of Gandalf, an internal KMS (Knowledge Management System). Designed to streamline knowledge-sharing, Gandalf serves as a central platform where team members can both disseminate and seek out vital company information.

My role encompassed a wide range of responsibilities, from full-stack development to the strategic migration of the backend architecture. This project not only enhanced my technical skills but also provided invaluable experience in transforming an internal tool into a commercial SaaS product.

Core Contributions & Feature Development

Initially built on a NodeJS and VueJS stack, my responsibilities included:

  • Developing a responsive and modern UI with VueJS and Vuetify.
  • Building a robust backend with NodeJS and managing database interactions with Sequelize and MSSQL.
  • Implementing diverse features like a skills portfolio, a secure external login system, content versioning, and admin management modules.
  • Working within an Agile/SCRUM framework using Azure DevOps for CI/CD.

Technical Migration to .NET

I actively participated in the strategic migration of the backend from NodeJS to a more scalable and robust C# .NET architecture. This crucial task involved:

  • Rewriting the application's business logic in C#.
  • Replacing the Sequelize ORM with Entity Framework Core (EF Core) for improved performance and type safety.
  • Ensuring a smooth and seamless data and logic transition.

Commercialization and Multi-Tenancy

As the project matured, I helped transform it from an internal tool into a commercial SaaS product. I played a vital role in this transition by:

  • Architecting and implementing a multi-tenant system, enabling the application to serve multiple clients from a single, secure instance.
  • Contributing to the marketing process by helping create promotional materials, including flyers and the design of the product's showcase website.
