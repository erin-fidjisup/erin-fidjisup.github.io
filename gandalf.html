Help in the development of many Feature in the KMS Gandalf

Project information

  • Category: Web Development
  • Client: Reboot Conseil
  • Project date: July 2023 - January 2024
  • Technology: NodeJS, VueJS (Vuetify), Sequelize, Azure DevOps, SCRUM, AGILE

At Reboot Conseil, I had the distinct opportunity to be a pivotal contributor to the development of Gandalf, an internal KMS (Knowledge Management System). Designed to streamline knowledge-sharing, Gandalf serves as a central platform where team members can both disseminate and seek out vital company information.

My primary areas of involvement in this project included:

  • VueJS: Spearheaded the development of a user interface using VueJS, leveraging Vuetify for a seamless and modern design.
  • NodeJS: Established a sturdy backend architecture using NodeJS, meticulously organized around controllers, services, and other foundational components.
  • Sequelize: Employed Sequelize for optimal database interaction, ensuring efficient data querying and management.
  • SCRUM Methodology: Played an instrumental role in the development process by adopting the SCRUM methodology, ensuring timely and effective product delivery.
  • MSSQL: Orchestrated the design and management of databases using MSSQL, maintaining relational integrity and peak performance through intricate table associations.
  • Azure Devops: Utilized Azure Devops for source code management, continuous integration, and deployment processes, ensuring streamlined development cycles.

Furthermore, I took the reins in rolling out several key features, such as:

  • Launching a competency centralization platform, aptly named the Portfolio.
  • Designing a secure login mechanism tailored for external users, enhancing platform security and trustworthiness.
  • Introducing a versioning system for posts, guaranteeing traceability and the evolution of shared information.
  • Implementing a backlog module for managing categories and users, thereby refining platform administration capabilities.
