Portfolio Details - Self hosting Website

Search for and deploy a self-hosted website solution

Project information

  • Category: Self self-hosted website using a VPS
  • Client: Reboot Conseil
  • Project date: October , 2023
  • Technology: VPS, Nginx web server, Azure CI/CD, Certbot, Azure AppInsight, domain name using OBH Cloud

This project was an ambitious endeavor to self-host websites on a Virtual Private Server (VPS), focusing on maximizing automation in deployment. Initially, there was a phase of research and comparison of solutions to understand the requirements. Four main solutions were considered: turnkey solutions, VPS, a Kubernetes cluster, or private cloud hosting. After this research phase, testing commenced on a VPS using an Nginx web server. The production phase involved deploying on the VPS using Azure CI/CD. A significant aspect of the project was automating the renewal of SSL certificates using Certbot. Monitoring was efficiently handled using Azure AppInsights. All domain names for the project were managed through OVH Cloud. A crucial part of this project was the development of a Python script for streamlining the deployment process. This script was instrumental in ensuring that the deployment of the solution was both efficient and effective.
