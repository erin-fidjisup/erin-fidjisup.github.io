BigData

Project information

  • Category: BigData project
  • Client: School Project
  • Project date: March, 2023
  • Technology: MongoDB Database, Talend, R script (ggplot2, readr, tidyverse), Python script, NodeJS

The project I worked on served as an introduction to the field of Big Data and involved the processing of medical data. Our objective was to handle a large dataset, perform data filtering, classification, and modeling. We utilized various technologies throughout the project, including Talend for data integration, Python for data filtering, and R and Node.js for interface development. The main tasks and components of the project included:

  • Data Integration with Talend: We leveraged Talend, a powerful data integration tool, to handle the large volume of medical data. Talend allowed us to extract, transform, and load the data from various sources into a unified format suitable for analysis.
  • Data Filtering with Python: To preprocess the medical data, we employed Python. Using Python's libraries and functionalities, we applied filtering techniques to refine the dataset, removing irrelevant or noisy data points, and ensuring data quality.
  • Data Classification: Once the data was preprocessed, we focused on implementing classification algorithms. We utilized machine learning techniques to categorize the medical data into different classes or groups based on specific attributes or patterns. This step enabled us to gain insights and discover meaningful patterns within the dataset.
  • Data Modeling with R: R, a statistical programming language, was used for data modeling and analysis. We leveraged R's extensive libraries and packages to perform advanced statistical analysis, generate visualizations, and build predictive models based on the processed medical data.
  • Interface Development with R and Node.js: We created an interactive interface to facilitate data exploration and visualization. R was used to generate visual representations of the analyzed data, while Node.js was employed to develop a web-based interface that allowed users to interact with the processed data, view visualizations, and access relevant insights.
