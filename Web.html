Website to find an intership

Project information

  • Category: Introduction to the web
  • Technology Used : HTML, CSS, Bootstrap, PWA, PHP, Javascript
  • Project date: June 2022

For this project, we had to create a web interface for an internship search site. This project, from scratch, asked us to create a graphic charter as well as the associated mockups to meet the specifications. Then, we realized a frontend in HTML/CSS/Javascript, using the Bootstrap library to manage the responsiveness of our interface. Finally, the backend of this application was realized in PHP

  • Graphic Charter and Mockup Creation: We started by defining a graphic charter for the interface. This involved selecting appropriate colors, typography, and visual elements to establish a cohesive and consistent design. We then created mockups, which served as visual representations of the interface layout and functionality, adhering to the graphic charter and meeting the specifications provided.
  • Frontend Development: Using HTML, CSS, and JavaScript, we developed the frontend of the web interface. We focused on implementing the mockup designs and ensuring a responsive layout using the Bootstrap library. This enabled the interface to adapt and display properly on different devices and screen sizes.
  • User Interface (UI) Design: During the frontend development phase, we paid special attention to the UI design, ensuring an intuitive and user-friendly experience. We implemented interactive elements, navigation menus, search functionality, and other features to enhance the usability of the interface.
  • Backend Implementation: The backend of the application was developed using PHP. We created the necessary server-side scripts to handle user inputs, process data, and interact with the database. This included implementing user authentication, internship search functionality, and managing user profiles and applications.
  • Integration and Testing: We integrated the frontend and backend components, ensuring seamless communication and functionality between them. We conducted thorough testing to identify and resolve any issues or bugs, ensuring the proper functioning of the web interface.
  • Deployment : Using hosting and domain name to make our site accessible
