Network Infrastructure from scratch

Project information

  • Category: Network Infrastructure
  • Client: School Project
  • Project date: March , 2022
  • Technology: VLANS, IPV6 tunnel, IP adressing, Networking, Cisco Packet Tracer

The project I undertook involved the creation of a network infrastructure from scratch using Packet Tracer simulation software. The objective was to design and configure a network with multiple sites, implementing different VLANs, and incorporating basic network concepts such as routing, IP addressing, VLANs, and IPv6 tunneling.

  • Network Design: We started by designing the network infrastructure, considering the different sites and their interconnections. This involved planning the layout of routers, switches, and end devices to ensure efficient and secure communication between sites.
  • IP Addressing: We assigned IP addresses to the network devices based on the requirements of each VLAN and site. This included choosing appropriate IP address ranges, subnetting, and configuring IP addresses on routers, switches, and end devices.
  • VLAN Configuration: We implemented VLANs to logically segregate the network into different broadcast domains, enhancing security and network management. Each site was assigned its respective VLANs, and VLAN tagging was configured on switches to allow for VLAN trunking between sites.
  • Routing: We set up routing protocols, such as OSPF (Open Shortest Path First) or EIGRP (Enhanced Interior Gateway Routing Protocol), to enable communication between different sites and VLANs. Routing tables were configured on routers to facilitate the exchange of routing information.
  • IPv6 Tunneling: To introduce IPv6 connectivity in the network, I implemented IPv6 tunneling techniques such as 6to4 or GRE (Generic Routing Encapsulation). This allowed for the transmission of IPv6 packets over an IPv4 network infrastructure.
  • Packet Tracer Simulation: Using the Packet Tracer software, I simulated the network infrastructure to test and validate its functionality. I performed various tests and scenarios to ensure proper routing, VLAN segmentation, and IPv6 connectivity.
